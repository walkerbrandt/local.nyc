How we select.

The rubric behind every event

The core test is temporal specificity. An event passes if removing the date makes it meaningless. An opening night passes. A gallery show on view doesn't. A single 35mm print screening passes. "Screenings ongoing" doesn't. We cross-reference sources — if a venue's event has been noted by Artforum, Screen Slate, or a comparable publication, it moves up.

What passes

  • Opening night / vernissage
  • One-night performance or event
  • Limited run — 3 nights or fewer
  • Screening + Q&A or director present
  • Pop-up dinner, one-night kitchen takeover
  • Debate, lecture, notable structured talk
  • Music — first show or rare booking
  • Ongoing exhibition with no opening event
  • Recurring weekly nights
  • Standard restaurant service

Cross-referenced by

  • Artforum events calendar
  • artnet.com/events — gallery openings
  • Screen Slate — film screenings
  • Event noted in editorial press
  • Venue Instagram (66 Greene, Pioneer Works)
  • Generic Eventbrite without primary source
  • Aggregators without original citation

Venue signal

The location matters. Galleries that get reviewed. Rep cinemas. Concept stores with a programming track record. Restaurants that are themselves a cultural moment. Bookshops and institutions with history. Not hotel ballrooms.

Before it's published

  • Confirmed date, address, and time
  • Real source URL — every event
  • Venue details verified
  • No source URL = not published
  • Unverifiable listings excluded
